Kesra Satu Data
Menu Utama

Ringkasan Data

Gunakan menu samping untuk memilih jenis data. Admin Desa dapat menambah dan mengedit data untuk desa sendiri. Verifikator dapat menghapus data dan mencari nama perorangan.

Panduan Aplikasi

  • Admin Desa dapat mengelola data desa masing-masing.
  • Verifikator dapat menghapus data dan mencari berdasarkan nama.
  • Semua daftar dapat diekspor ke Excel, Word, dan PDF.
  • Gunakan tombol "Persentase Semua Data" untuk melihat distribusi data.
